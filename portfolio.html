Collect Birthstones Game

Collect valuable birthstones from one of the four randomly generated of the twelve monthly birthsones until you match the value of the Jackpot or go over. Keeps track of wins and losses. Uses jQuery to dynamically create the four birthstones and assign them a value for each round. jQuery is also used to dyanmically update the total value, populate user messages and update the score.

When's the Next Train

Emulates the big board at the train station. Using moment.js for time calucations and Google's Firebase to store inforation about each train, display the name of the train, where it originated, when the next train is scheduled to arrive and how many minutes away it is. Allow the user to add or delete a train or modify an existing train.
